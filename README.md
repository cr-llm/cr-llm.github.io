# Dagstuhl Seminar 26152: Causality and Large Language Models

Website for the Dagstuhl Seminar "Causality and Large Language Models: Opportunities to Advance Causal Reasoning" (April 7-10, 2026).

## About

This repository hosts the website for Dagstuhl Seminar 26152, which explores the intersection of causal machine learning and large language models.

## Local Development

To run the site locally:

1. Install Ruby and Bundler
2. Clone this repository
3. Run `bundle install`
4. Run `bundle exec jekyll serve`
5. Visit `http://localhost:4000` in your browser

## Deployment

The site is automatically deployed to GitHub Pages using GitHub Actions when changes are pushed to the main branch.

## Structure

- `index.md` - Home page
- `organizers.md` - Information about the seminar organizers
- `schedule.md` - Seminar schedule and program
- `participants.md` - List of participants
- `location.md` - Venue and travel information
- `resources.md` - Reading materials and related resources
- `_config.yml` - Jekyll configuration
- `_layouts/` - HTML templates
- `_includes/` - Reusable HTML components
- `assets/css/` - Stylesheets
